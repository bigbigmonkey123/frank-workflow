# Workflow

## Compact Path

```text
Inspect -> Minimal change -> Local QA -> Review -> Report
```

## Full Gated Path

```text
Research -> Plan -> Pre-dev Review -> Human FTW if needed -> Implement -> QA -> Post-dev Review -> Commit/PR -> Docs closeout
```

## Multi-agent Path

```text
Scout -> Split tasks -> Developer workers -> Reviewer review -> Fix -> Lead merge -> QA -> Final review
```
